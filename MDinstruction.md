# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его (*).
Пример : *вот так*

Чтобы выделить текст полужирным необходимо его обрамить двойными (*). 
Пример : **вот так**
